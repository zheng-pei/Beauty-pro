
这是一个微信端的美容预约的项目，希望大家可以参考，只能用于学习参考哦！ 文件的目录：
     Lib---------thinkphp3.1的核心文件

     weiwin--------前台和后台的逻辑代码
     data---------/Conf------里面的Conf文件夹是存放配置的文件------/System----存放的是后台配置文件，
                      -------里面的db.conf是数据库配置文件
     tpl----static 静态文件,wap前台的模板文件，System是后台的模板文件
     Extend------是扩展包插件
     Uploads 上传图片的文件
     index.php 入口文件
     admin.php 后台入口的文件
     weipay-------------微信支付的接口代码
     wxpay-------------微信支付的日志

默认的数据库名beauty.sql ，用户名是root，数据库密码为空
